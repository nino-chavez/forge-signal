## Key Strategic Context

### The Opportunity

- **25,000 WooCommerce stores migrate annually**, 80% default to Shopify
- **Target capture:** 5% of migrations = $5.65M ARR (base case)
- **Investment:** $190K Phase 0 validation → 29.7x ROI
- **Partnership:** Stealth-mode collaboration with WPEngine (8,000 target stores)

### The Value Proposition

**"Keep WordPress. Upgrade Commerce."**

Commerce.com positions BigCommerce + Feedonomics + Makeswift as the "WordPress-native commerce upgrade" — enabling merchants to preserve their WordPress content investment while replacing the failing WooCommerce commerce layer.

### Critical Risks (from Red Team Audit)

- ⚠️ **WPEngine partnership is unvalidated** — no evidence they want this
- ⚠️ **Competitive response underanalyzed** — Automattic could counter
- ⚠️ **Customer acquisition depends on WPEngine** — no fallback plan

### Role Archetype

**Product Director / GM for Composable Commerce Solutions**

- **P&L Ownership:** $2.25M - $5.65M ARR target, $850K budget
- **Cross-Product Delivery:** Integrate BigCommerce + Feedonomics + Makeswift (post-acquisition silos)
- **Organizational Bridge:** Navigate acquired company teams, build Tiger Team
- **POC → Production Discipline:** Ship pilots in 12 weeks, scale to GA
