Data Classification in Flow
Public Data
  • • Product catalog
  • • Pricing (non-personalized)
  • • Availability
  • • Shipping options
Consented Data
  • • Customer preferences
  • • Purchase history
  • • Saved addresses
  • • Session context
Protected Data
  • • Payment tokens (SPT)
  • • Full payment details
  • • PII (SSN, DOB)
  • • Health/financial data
Key Integration Points
Synchronous
  • • Product search (< 200ms)
  • • Price calculation (< 100ms)
  • • Inventory check (< 150ms)
  • • Payment auth (< 3s)
Asynchronous (Webhook)
  • • Order confirmation
  • • Shipping updates
  • • Delivery confirmation
  • • Return status
Escalation Triggers
Human Handoff When:
  • • Confidence score < 0.7
  • • Order value > $500
  • • Payment mandate exceeded
  • • Customer frustration detected
  • • Complex return/dispute
  • • Explicit escalation request
