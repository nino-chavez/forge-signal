1 / 10 Agentic Commerce
01 Strategic Context Data is the new storefront
Consumer journeys now begin in AI conversations, not search boxes
AI agents aggregate, compare, and transact across retailers—the storefront has moved upstream. The retailer that controls the data layer controls the transaction.
The Shift

AI-powered shopping layers position themselves between retailers and customers. Without structured data and agent protocols, you become invisible to AI discovery.

The Risk

AI agents aggregate and compare products across retailers—you're no longer competing on your own site. If agents can't understand your data, they won't recommend you.

The Strategy

Build the infrastructure layer AI agents depend on. Optimize for GEO, implement commerce protocols, and become the authoritative source AI trusts.

Agent Ecosystem Who connects to your systems
Owned Agents
Your AI assistants—shopping helpers, gift finders, support bots. Full data access, complete control.
External Agents
Third-party AI (ChatGPT, Claude, Gemini). Limited data access, protocol-mediated transactions.
Partner Agents
Co-developed with vendors (brands, manufacturers). Contracted data sharing, co-branded experiences.
Customer Agents
Consumer's personal AI acting on their behalf. Mandate-based permissions, customer-consented access.

07 Agentic Design Patterns Building blocks for agent orchestration
Workflow Patterns Structured

Prompt Chaining

Sequential LLM calls where each output feeds the next. Decomposes complex tasks into fixed subtasks.

When: Tasks cleanly separable into sequential steps

Routing / Handoff

Classify input and direct to specialized handlers. Different categories get different processing.

When: Customer service triage, cost-optimized model selection

Parallelization

Run multiple LLM operations simultaneously. Sectioning (independent) or Voting (consensus).

When: RAG query decomposition, multi-perspective analysis
Agentic Patterns Dynamic

ReAct (Reason + Act)

Iterative loop: Thought → Action → Observation → Repeat. Agent reasons about tools, executes, observes.

When: Complex queries requiring tool use, transparent reasoning chains

Planning (Orchestrator-Workers)

Central planner decomposes task into subtasks, delegates to specialists, synthesizes results.

When: Multi-step reasoning, unpredictable decomposition

Multi-Agent + Human-in-the-Loop

Specialized agents collaborate; pause at checkpoints for human review on high-stakes actions.

When: Financial transactions, regulatory compliance, complex returns

08 Multi-Agent Orchestration Team coordination patterns

Sequential

Agents execute in predefined linear order—output from one becomes input for next.

When: Structured pipelines: extract → clean → analyze → report

Parallel

Multiple agents work simultaneously, outputs synthesized into consolidated response.

When: Concurrent analysis, diverse perspectives, latency reduction

Coordinator (Hub & Spoke)

Central coordinator analyzes requests, routes to specialists, aggregates responses.

When: Adaptive routing, dynamic task assignment, customer service

Hierarchical

Multi-level hierarchy where parent agents decompose tasks for subordinates recursively.

When: Ambiguous problems, deep decomposition, research projects

Review & Critique

Generator agent creates; critic agent evaluates against criteria and approves/rejects/returns feedback.

When: High-accuracy requirements, code review, content quality gates

Swarm

Agents collaborate with all-to-all communication, iteratively refining through shared findings.

When: Highly complex problems, multi-constraint optimization
Start simple—add complexity only when simpler solutions fail
Begin with optimized single LLM calls, add workflow patterns for predictable multi-step tasks, use ReAct as default for tool-using tasks. Combine patterns as needed: Planning + Tool Use, Multi-Agent + Reflection, ReAct + Human-in-the-Loop.

09 Glossary of Key Terms Essential terminology for agentic commerce
Protocol Terms
ACP (Agentic Commerce Protocol)

OpenAI's standard for AI agent shopping and checkout

AP2 (Agent Payments Protocol)

Google's mandate-based purchase permissions

MCP (Model Context Protocol)

Anthropic's universal tool/data integration

Mandate

Permission rules defining agent action limits

SPT (Secure Payment Token)

Limited-use token for agent transactions

AI/ML Terms
RAG (Retrieval-Augmented Generation)

AI looks up real data before answering

Embedding

Numeric representation of meaning for search

Hallucination

AI confidently stating incorrect information

Groundedness

Responses anchored in actual data sources

Tool Calling

LLM invoking external APIs during conversation

Commerce & GEO

SEO for AI agents (ChatGPT, Gemini, etc.)

Citation Share

% of AI answers citing your brand as source

Unified customer profile and history

ATP (Available to Promise)

Real-time inventory available to sell
